# Ders Programı - Masaüstü Uygulaması

Haftalık ders programı hazırlama ve yönetme uygulaması.

## Geliştirme

### Web uygulaması olarak çalıştırma
```bash
npm run dev
```

### Electron uygulaması olarak çalıştırma
```bash
npm run electron:dev
```

## Build

### Tüm platformlar için build
```bash
npm run electron:build
```

### Windows için build
```bash
npm run electron:build:win
```

### macOS için build
```bash
npm run electron:build:mac
```

### Linux için build
```bash
npm run electron:build:linux
```

Build edilen dosyalar `release` klasöründe olacaktır.

## Özellikler

- ✅ Haftalık ders programı oluşturma
- ✅ Ders ve konu yönetimi
- ✅ PDF/Print desteği
- ✅ LocalStorage ile veri saklama
- ✅ Masaüstü uygulaması desteği (Electron)
